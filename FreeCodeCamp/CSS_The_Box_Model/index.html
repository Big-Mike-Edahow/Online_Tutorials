CSS Box Model Diagram


1st Box Model Layer: Content

The content of the box,
where text and images appear.

2nd Box Model Layer: Padding

Clears an area around the content.
The padding is transparent

3rd Box Model Layer: Border

A border that goes around
the padding and content.

4th Box Model Layer: Margin

Clears an area outside the border.
The margin is transparent
