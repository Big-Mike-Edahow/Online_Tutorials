Using Python's http.server module


Serving up static HTML and CSS


It is really neat how far technology has advanced in the past several years.


If a developer wants to set up a simple HTTP server to serve up some static HTML and CSS files, all that they need to do is import the http.server and socketserver modules from the Python standard library. With just a few simple lines of Python code you have a web server to play around with.
